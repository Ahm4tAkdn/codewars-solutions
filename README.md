# Codewars C# Çözümleri

Bu repository, Codewars platformunda çözdüğüm ve
staj başvurularında paylaşmak üzere seçtiğim C# problemlerini içermektedir.

## Amaç
Bu çalışmanın amacı:
- Algoritmik düşünme yeteneğimi göstermek
- C# diliyle problem çözme yaklaşımımı sergilemek
- Okunabilir ve düzenli kod yazma alışkanlığımı yansıtmaktır

## İçerik
Bu repository içerisinde:

- 5 kyu seviyesinde 1 adet problem
- 6 kyu seviyesinde seçilmiş problemler

Her problem için:
- Ayrı klasör yapısı
- Türkçe açıklama içeren README dosyası
- Temiz ve anlaşılır C# kodu

## Kullanılan Teknolojiler
- C#
- .NET (Console / Class Library yapısı)
